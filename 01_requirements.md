# Product Requirements: FocusDo

## 1. App Overview
FocusDo is a minimal task management application designed for single users to track daily tasks without distractions.

## 2. Target Audience
Individual users who need a simple list, not complex project management features.

## 3. Core Features (MVP - Minimum Viable Product)
1. **User Authentication:**
   - Users can sign up with email/password.
   - Users can log in and log out.
2. **Task Creation:**
   - Users can input a text description to create a new task.
   - Tasks are added to the top of the list by default.
3. **Task Management:**
   - Users can mark a task as "Complete" (checkbox).
   - Users can delete a task permanently.
   - Users can edit the text of an existing task.
4. **Task Viewing:**
   - Users can see a list of all active tasks.
   - Users can filter to see "Completed" tasks separately.

## 4. Out of Scope (Do NOT Build)
- No shared lists or teams.
- No push notifications or email reminders.
- No complex due dates or calendars.
- No "Dark Mode" (keep UI strictly black/white for now).
